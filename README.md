# Games with libGDX

RPG game like Zelda using libGDX Library


## Prerequires

1. Git 2.6+
2. Gradle 2.11+
3. Java 8+


## How to Play

Clone

```
git clone https://github.com/humbertodias/game-gizelda-libgdx.git
```

Inside the folder

```
cd game-gizelda-libgdx/src
```

Run

```
gradle desktop:run
```
